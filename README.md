# 📥 INSTALLATION

## 📂 STEP 1

Download the latest release.

---

## 📂 STEP 2

Locate your Roblox installation folder.

Open:

```text
Roblox
└── Versions
    └── version-xxxxxxxxxxxx
        └── PlatformContent
            └── pc
                └── textures
📂 STEP 3

Open the Dark Textures folder you downloaded.

📂 STEP 4

Copy all files inside the Dark Textures folder.

📂 STEP 5

Paste them into:

PlatformContent\pc\textures
📂 STEP 6

When Windows asks:

Replace the files in the destination?

Click:

✅ REPLACE ALL
📂 STEP 7

Restart Roblox.

🎉 DONE!

Dark Textures should now be installed.

🛠 INSTALLATION METHOD USED

The installation was completed by:

✅ Downloading the Dark Textures folder

✅ Clicking "Open Roblox File Location"

✅ Clicking "Open Roblox File Location" again

✅ Opening:

PlatformContent
└── pc
    └── textures

✅ Copying every file from the Dark Textures folder

✅ Pasting them into the textures folder

✅ Clicking:

Replace All

✅ Restarting Roblox

🔄 AFTER ROBLOX UPDATES

Roblox updates may overwrite texture files.

If textures disappear:

Open the newest Roblox version folder.
Navigate to:
PlatformContent\pc\textures
Paste Dark Textures again.
Click Replace All.
Restart Roblox.
❌ UNINSTALL
Option 1

Reinstall Roblox.

Option 2

Restore your original texture backup.
